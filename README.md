# Remedify
Welcome to Remedify!
- Landing page: https://remedify.vercel.app/
- Blog: https://remedify-blog.vercel.app/
## Tools
- Expo
- Azure API
- OpenAI API
- Kitten UI
## How to run
1. Clone repo
2. `npm i`
3. `npx expo start`
4. Press w in terminal to open web version
- **Note: OpenAI API features will not work without valid key in .env**
